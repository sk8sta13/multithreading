# Multithreading

Esse programa faz duas consultas de CEP em duas API diferentes (brasilapi e viacep), a API que tiver a resposta mais rápida dentro de 1 segundo é a que será exibida no terminal.

## Teste

Para executar um teste:

```
go run . 01153000
```
